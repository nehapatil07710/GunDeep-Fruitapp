🛒 FruitCart – Complete Functional Description
📌 Overview

FruitCart is a React-based e-commerce web application that allows users to browse, search, and purchase fruits online with a smooth and modern UI. The application integrates Firebase Authentication, cart management, and real-time user interaction features to simulate a real-world shopping experience.

🔐 1. Authentication System
✅ Login (Email & Password)
Users can log in using registered email and password
Firebase Authentication is used for secure login
Validation included:
Invalid email handling
Wrong password error
After login:
User data stored in LocalStorage
Redirected to main/home page
✅ Signup (Create Account)
New users can register with:
Name
Email
Password
Firebase handles account creation
Error handling:
Email already exists
Weak password
After successful signup:
Success message shown
Redirect to login page
🔵 Login with Google
Users can sign in using Google account
Firebase Google Authentication integration
One-click login (no need to enter password)
Improves user experience and reduces friction
🏠 2. Home Page / Dashboard
Displays available fruit products
Clean and attractive UI
Includes:
Product cards
Price display
Add to cart option
🔍 3. Search Functionality
Users can search fruits by name
Real-time filtering of products
Improves usability and quick navigation
🛒 4. Cart System
✅ Add to Cart
Users can add fruits to cart
Cart updates dynamically
✅ Cart List
Displays all selected products
Shows:
Product name
Quantity
Price
✅ Cart Total
Automatically calculates total amount
Updates in real-time
🧾 5. Bill Generation System
Generates a final bill based on cart items
Includes:
List of items
Total price
Acts like a mini checkout system
📲 WhatsApp Bill Feature (🔥 Unique Feature)
Users can send their bill via WhatsApp
Generates a formatted message like:
Order Details:
Apple - ₹100
Banana - ₹60
Total: ₹160
Opens WhatsApp with pre-filled message
Useful for:
Placing orders
Sharing bill
📜 6. Order History
Stores previous orders
Displays past purchases
Helps users track:
What they bought
Total amount
🌙 7. Dark Mode
Toggle between:
Light mode ☀️
Dark mode 🌙
Improves UI experience
Better usability at night
💾 8. Local Storage Usage
Stores:
User login data
Cart items
Order history
Ensures data persistence even after refresh
🎨 9. UI / UX Features
Glassmorphism design (modern UI)
Responsive layout (mobile + desktop)
Smooth hover effects
Clean forms and navigation

Built using React (Vite)
Uses:
React Hooks (useState)
Component-based architecture

Authentication
CSS used for styling and animations
